# JARVIS Weather Plugin

A weather plugin for the JARVIS AI Assistant.
Provides real-time weather and forecasts
for any city in the world.

## Features
- Current weather conditions
- Temperature, humidity, wind speed
- UV index and pressure
- Multi-day forecast
- No API key required

## Installation
Say to JARVIS:
"install https://github.com/YOURUSERNAME/jarvis-weather-plugin"

## Commands
- "what's the weather in Istanbul?"
- "weather forecast for Ankara"
- "is it raining in London?"

## Requirements
- requests library (auto-installed)
